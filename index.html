Forms

Send the Data of the forms to Formspree.com
<form action="URL OF FORMSPREE" method="POST">
Then insert inside Input for users, Attached with a label for each input:
<label for="user-name">User Name:</label>
<input type="text" id="user-name" name="user-name">
You can also add "value":
<input type="text" id="user-name" name="user-name" value="Write your User-Name Here">
Then add button to send the data to Formspree:
<button>send</button>
Or:
<input type="submit" />
