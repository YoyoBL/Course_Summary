Apply CSS to HTML

There are 3 ways to apply CSS styling to the HTML:

  1. Styling inside the element tag itself:
    <h1 style="color: black; font-size: 15px;"></h1>
  2. Styling inside the HTML file Using <style> tag:
     <style>
      h1 {
        color: black;
        font-size: 15px;
      }
    </style>
    You should place the <style> tag in the <head> tag
  3. Linking to external CSS file:
     <link rel="stylesheet" href="./location of file">
    This is the recommended method to style entire web-sites.

CSS Selectors

Selectors are used to target specific parts of the HTML code.
These are the selectors we learned so far:

  1. #id = ID selector. Unique, there can not be 2 elements with the same ID.
  2. .class = Class selector. Not unique, can be used on elements as many time as needed.
  3. * = All elements selector.
  4. space/רווח = All child's selector.
  5. > = Direct child selector.
  6. , = Or selectors. If we want to apply the same style on multiple parts.

Selectors combinations 

  • div {
   selects all Div's
}

h1 {
   Selects all h1's
}  
    
  • div#hello {
   Selects only the div with the id="hello"
        There can be only 1 because it's an id.
}
   
  • div.hello {
   Selects only the div with the class="hello"
        There can be multiple because it's a class.
}
   
  • div #hello {
   Selects only the child of div with the id="hello" 
} 
   
  • div .hello {
   Selects only the child's of div with the class="hello" 
}
   
  • div > #hello {
   Selects only the direct child of div with the id="hello"         
}
