Link CSS file to HTML

There are 3 ways to link your CSS file to the HTML:

  1. Styling inside the HTML tag: <h1 style="color: black; font-size: 15px;"></h1>
  2. Styling inside the HTML file Using <style> tag: <style> h1 { color: black; font-size: 15px; } You should place the <style> tag in the <head>> tag </style>
  3. Linking to external CSS file: <link rel="stylesheet" href="./location of file"> This is the recommended method to style entire web-sites.
